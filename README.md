# 🌿 Cladograma Interativo de Macrófitas Marinhas do Brasil

Bem-vindo ao repositório do **Cladograma Interativo de Macrófitas Marinhas**!  
Este projeto visa facilitar a visualização, exploração e divulgação científica da biodiversidade de macrofitas marinhas do Brasil, reunindo dados taxonômicos, scripts e uma visualização interativa moderna.

## 🔗 Acesse o Portal Interativo

👉 [Visualização Interativa no GitHub Pages](https://caetanoronan.github.io/Cladograma_macrofitas/)

---

## ✨ Funcionalidades

- **Visualização interativa** com D3.js (HTML, CSS, JavaScript)
- Filtros dinâmicos, zoom e navegação intuitiva
- Dados taxonômicos em formatos JSON e CSV
- Scripts Python para processamento e conversão de dados
- Pronto para publicação via GitHub Pages

---

## 📁 Estrutura do Repositório

- `index.html` — Página principal da visualização interativa
- `cladograma_interativo_macrofitas_funcional_filtros_zoom_teste_pronto.html` — Versão funcional com filtros e zoom
- `cladograma_interativo_melhorado.html` — Versão aprimorada do cladograma
- `mapa_google_macrofitas.html` — Mapa de distribuição das macrofitas
- `*.json`, `*.csv` — Dados taxonômicos e de distribuição
- `*.py` — Scripts para manipulação e conversão de dados

---

## 🚀 Como usar

1. Acesse o portal interativo pelo link acima.
2. Explore o cladograma, utilize filtros e navegue pelas relações filogenéticas.
3. Para rodar scripts ou modificar dados, clone o repositório e utilize os arquivos Python conforme necessário.

---

## 📚 Sobre o projeto

Este projeto foi desenvolvido para apoiar pesquisas, ensino e divulgação científica sobre a diversidade de macrofitas marinhas no Brasil, especialmente no contexto do PPGOceano/UFSC.
